# java-cli-gradle-sirixdb-simple

## Description
Reads from json and poopulates a database.

## Tech stack
- java
- gradle
  - sirixdb

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
